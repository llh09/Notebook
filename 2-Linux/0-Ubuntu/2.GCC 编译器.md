# GCC 编译器

## 1. 命令

`gcc [选项] [文件]`

选项：

-c：只编译不链接，输出为 .o 文件

-o：指定输出文件名，默认输出 a.out

-g：添加调试信息，使用 GDB 等调试工具的话需要加入此项

-O：对程序进行优化

-O2：比 -O 更大幅的优化

## 2. GCC 编译流程

预处理 --> 编译 --> 汇编 --> 链接

预处理：展开所有的头文件，替换程序中的宏、解析条件编译并添加到文件中

编译：将经过预编译处理的代码编译成汇编代码

汇编：将汇编语言文件编译成二进制目标文件

链接：将汇编出来的多个二进制目标文件链接在一起，形成最终的可执行文件（包括动态库，静态库等等）
